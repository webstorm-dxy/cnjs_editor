# cnjs_editor

CNJS以及多种语言的编辑器。

### 代码文件结构
```text
./src
├── App.vue       // 应用程序组件(目前只有这一个启用，其他的都是系统自带的）
├── assets
│   └── logo.png
├── components
│   └── HelloWorld.vue
├── main.js
├── plugins
│   └── element.js
├── router
│   └── index.js
├── store
│   └── index.js
└── views
    ├── AboutView.vue
    └── HomeView.vue

```

## Project setup
```
yarn install
```
